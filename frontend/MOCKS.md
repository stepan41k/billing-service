## Удаление моков (когда бэк готов)

```bash
# 1. Удалить папку с моками
rm -rf src/mocks

# 2. Удалить пакет
npm uninstall msw

# 3. Удалить 4 строки из src/main.tsx (между // MOCK и // END MOCK)

# 4. Удалить VITE_MOCK из .env.local

# 5. Удалить ARG VITE_MOCK из Dockerfile
```

### Что трогать при удалении

| `src/main.tsx` | ✅ 4 строки |
| `src/mocks/` | ✅ удалить целиком |
| `Dockerfile` | ✅ убрать ARG |
| `docker-compose.yml` | поставить флаг false |
